To compile this project,
1. Export Unity project Code_Samples/Unity Projects/UnityChan to Android project
2. Import it
3. Set that Android project as library under Project/Properties/Android/Library-> check "Is Library"
4. Add it to this project's library reference under Project/Properties/Android/Library
